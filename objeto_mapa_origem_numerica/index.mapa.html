Origem dos números

Sistema de numeração babilônico

O sistema de numeração babilônico é um sistema antigo de representação numérica desenvolvido pela civilização babilônica na Mesopotâmia, datado de 4.000 anos atrás. Essa civilização usava um sistema posicional baseado em 60, também conhecido como sistema sexagesimal.

Esse sistema apresenta apenas dois símbolos, que se repetem conforme vão sendo adicionados, gerando um novo número.

Sistema de numeração egípcio

O sistema de numeração egípcio é um sistema antigo de representação numérica, utilizado pela civilização egípcia e datado de cerca de 5.000 anos atrás. Esse sistema é considerado um sistema aditivo, pois os números são formados pela adição de símbolos com valores específicos.

Sistema de numeração maia

É um sistema matemático datado de 1.500 anos atrás, desenvolvido pela antiga civilização maia, que habitou principalmente a região da Mesoamérica (atualmente partes de México, Guatemala, Belize, Honduras e El Salvador).​ Esse sistema apresenta símbolos de pontos, traços e conchas, com representação posicional, ou seja, a representação do valor é determinada pela posição dos símbolos.

Sistema de numeração romano

É um sistema originário da Roma Antiga, desenvolvido há mais de 2.000 anos, desenvolvido como uma forma de identificação matemática simples. Esse sistema é composto de apenas sete símbolos, que correspondem a letras maiúsculas do nosso alfabeto. Atualmente é muito utilizado para identificar séculos, capítulos de livros, horário no mostrador de relógios etc. ​
